# Bali 2026 Trip

Публичная семейная витрина маршрута на Бали в октябре-ноябре 2026.

URL после деплоя GitHub Pages:

```text
https://timurgromov.github.io/bali-2026-trip/
```

## Стек

- Astro
- TypeScript
- Leaflet + OpenStreetMap
- GitHub Pages
- Один источник маршрута: `src/data/trip.json`

## Команды

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Правило данных

Маршрут, карта и дневной план не редактируются отдельно. Любое изменение баз, дат, переездов, точек карты, описаний или порядка маршрута сначала вносится в `src/data/trip.json`, после чего сайт пересобирается.

Маршрут считается по переездам и точкам проживания: 5 точек проживания и 4 основных переезда между ними. Повтор района всё равно считается новой точкой, если это новый блок жилья.

После любых изменений контента или верстки нужно просмотреть всю страницу на desktop и mobile, а не только изменённый блок.

В публичный repo нельзя добавлять PNR, паспорта, номера билетов, точные приватные адреса, скриншоты бронирований, PDF-билеты и другие приватные документы.
