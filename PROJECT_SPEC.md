# Project Spec - Bali 2026 Trip Site

## Purpose

Сделать отдельный публичный сайт маршрута семейной поездки на Бали, который можно отправить семье одной ссылкой. Сайт заменяет Notion как основную витрину маршрута.

## Scope v1

- Карта Бали с цифровыми пинами.
- Линия основного маршрута и пунктирные линии optional/day-trip направлений.
- Базы проживания по датам.
- День-за-днём программа.
- Переезды: время, расстояние, транспорт и окно выезда.
- Человеческие описания всех мест без непонятных голых названий.

## Non-goals v1

- Админка в браузере.
- Авторизация.
- VPS.
- Хранение приватных документов.
- Публикация на бизнес-домене пользователя или в коммерческих репозиториях.

## Source Of Truth

`src/data/trip.json` - единственный источник данных для текста, карты и логистики.

## Safety

Сайт публичный. Не добавлять:

- PNR;
- номера паспортов;
- номера билетов;
- точные адреса жилья сестры;
- телефоны;
- PDF-билеты;
- скриншоты бронирований;
- банковские данные.
